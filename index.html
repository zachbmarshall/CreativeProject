The Multi-App Website

I wanted to build a website that wasn't just a one-and-done school project that I would never touch again. I wanted to use this creative website project as an opportunity to build out a website that I will get some use out of.

That's why I built this Multi-App website. I started building out different tabs for different applications that I would use outside of a school setting. These are semi-random apps that do no have anything to do with each other at all, but they are applications that I would use. I have two ideas so far for these applications:


  • Score Board:

    This app takes a certain number of players and keeps track of their scores for them. It automatically rearranges in decending order by score. This is for those inconvenient times that you are playing a game and want to keep track of the score, but you don't want to write it down and keep track by hand.

  • Disney Ride Times

    This app gets the ride times at Disney Land and shows you the current wait times and then allows you to compare the current ride wait times to historical wait times. That way you can see when is the best time to visit the Happiest Place on Earth and when to ride each ride.
